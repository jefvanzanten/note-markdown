# note-markdown-sticky

Sticky note variant waar elke note een eigen window krijgt.

Gedrag:

- `+` maakt een nieuwe note en opent direct een nieuw sticky-window
- `Save` slaat de actieve note in dat window op
- `x` sluit dat window en verwijdert de bijbehorende note uit de sessie
- bij unsaved/dirty note verschijnt een confirm bij sluiten
